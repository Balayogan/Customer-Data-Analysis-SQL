# Customer Data Analysis | SQL

## Project Overview

This project analyzes customer transactions from the Global Superstore dataset using MySQL. The objective was to identify sales trends, customer behavior, product performance, and profitability insights through SQL-based analysis.

## Tools Used

- MySQL
- SQL
- Python
- Pandas
- SQLAlchemy

## Data Preparation

The dataset was imported into MySQL using Python and Pandas after identifying row discrepancies during the initial import process.

Data cleaning included:

- Converting Sales and Profit fields from text to numeric format
- Converting Order Date and Ship Date to DATE format
- Validating row counts between CSV and SQL tables

## Key Findings

- Western Europe generated the highest sales revenue.
- Technology was the highest revenue-generating category.
- Consumer customers contributed the highest sales.
- Apple Smart Phone, Full Size was the top-selling product.
- Several products generated losses despite strong sales volume.
